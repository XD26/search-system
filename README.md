## Описание

Система позволяет находить рецепты по:
- **текстовому запросу** (например, «chicken soup»);
- **загруженному изображению** (фотография блюда).

В основе системы лежат:
- **CLIP (ViT-B/32)** — модель для получения векторных представлений изображений и текстов;
- **FAISS (IndexFlatIP)** — библиотека для эффективного векторного поиска;
- **FastAPI** — веб-фреймворк для REST API и пользовательского интерфейса.

## Установка и запуск

### Требования
- Python 3.9+
- 8+ ГБ оперативной памяти (рекомендуется 16 ГБ)

### 1. Клонирование репозитория
```
git clone https://github.com/ваш_логин/recipe-search-system.git
cd recipe-search-system
```
### 2. Установка зависимостей
```
pip install -r requirements.txt
```
### 3. Подготовка данных
Для работы системы необходимо скачать датасет с Kaggle:
Название: Food Ingredients and Recipes Dataset with Images
Источник: Kaggle — Food Ingredients and Recipes Dataset with Images

После скачивания разместите файлы следующим образом:
CSV-файл с рецептами: data/Food Ingredients and Recipe Dataset with Image Name Mapping.csv
Изображения: data/Food Images/

### 4. Запуск сервера
```
python main.py
```
### 5. Открыть в браузере
Перейдите по адресу: http://localhost:8000
